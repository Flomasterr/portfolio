Technologies and principles that i use in my work

Html
Html with BEM metodology. Semantics, although this is a subjective thing, i try to use it where it makes sense. Pug preprocessor for Html templates.
Css
For Css i use Scss preprocessor, it allows me to use mixins and also speeds up and simplifies work with Css.
JavaScript
I use pure JavaScript, when need to add functionality like slider, tabs, spollers, gallery, tips, parralax, show/hide text etc i use modules/libraries for it. But the goal is to use as less as possible unnecessary stuff to make website faster.
Gulp + Webpack

Gulp + Webpack allows automate useful tasks such as:

  • Compression and optimization of images converting them into webp format
  • Compression and optimization of fonts converting them into woff2 format
  • Compression and optimization of css files converting them from scss and adding vendor prefixes for cross-browser support
  • Compression and optimization of js files
Adaptivity
I always make websites adaptive, even if the designer didn't do it in his design. This means that the website can be opened from any device on any screen width and it will not break.
Stability
When developing a website i take into account the possibility of changing/adding content in the future, this approach makes the website more stable and reliable. For example, if you want to add more items to the menu, more text in the text block, or change the image, the website will not break.
Interactivity
All interactive elements(elements with which the user can interact) will have hover effects.
